Datamaskinens oppbygging og virkemåte

Prosessor

Prosessoren er hjernen i datamaskinen.
Hastigheten på en CPU er mellom 2 - 4 GHz for nye maskiner.
CPU står for Central Processing Unit, og er det primære elementet
som gjennomfører damtamiskens hovedfunksjoner.

Intel

Intel er en amerikansk produsent av mikroprosessorer.
Firmaet ble grunnlagt 18. juli 1968
Selskapet var størst innen halvlederteknologi til det ble forbigått
av Samsumg i 2017. Første prosessor som ble laget var intel 4004 CPU, en 4 bits cpu.
4 bit gir 2^4 som største tall=16
For 5-6 år siden var standar størrelse 32 bit (tall på 4294967296)
Dagens CPUer er 64bit. Dagens CPU kan jobbe med tall på 2^64 = 18446744073709551616

AMD

AMD (Advanced Micro Devices) laget sin første CPU 8080 AM9080ADC
(En kopi av intel 8080)

ARM

Opprinelig Acorn (Advanced) RISC (Reduced Instructed Set Computer)
Machine. RISC CPUer har få og enkle instruksjoner (i motsetning til Intel).
Hver instruksjon skal ta minst mulig tid.
Kompliserte beregninger består av mange instrusksjoner.
Brukes i mobiler (nesten alle?)

RAM og ROM

RAM er en forkortelse som betyr Random Access Memory

Lagring

Standard lagringsmetode i dag er SSD. Tidligere (for få år siden) var HDD var HDD det mest brukte.
